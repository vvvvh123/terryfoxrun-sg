3. Current Benchmark

Keep what works, improve ownership and reuse

Reference behavior from current public materials

  • 2025 public listing shows 30 Nov 2025, Angsana Green ECP, 7am, 5km and 10km, and no entry fee.
  • Shirt and donation registration closes before event day, with pickup on a separate weekend.
  • FAQ states there are no race packs, bibs, timing chips, winners, or prizes.
  • No T-shirt size exchanges; QR confirmation can be used by a designated collector.

Sources: JustRunLah 2025 listing, TerryFoxRun.sg FAQ 2024, and existing registration form reference.

What the new site improves

  • Organizers control event setup, field requirements, inventory, announcements, and exports.
  • Participants get a clear dashboard and reusable pickup code instead of relying only on email.
  • Corporate shirt orders become a first-class flow for companies and larger groups.
  • Each year starts from a cloned event configuration rather than a vendor rebuild.
  • Past event history, transaction history, and mass email live in the same admin portal.

10. Approval + Next Steps

Approve the direction, then build in phases

Recommended MVP

  • Public homepage, event details, registration, corporate orders, dashboard, and past runs.
  • Stripe card, Apple Pay, and Google Pay for individual registrations.
  • Admin-configured yearly event setup, shirts, inventory, announcements, pickup, and exports.
  • Build once and reuse every year, with past event history and transaction history preserved.
  • Mass email plus dashboard communication from the website admin portal.

Phased implementation

Phase 1
Approval, copy, assets, and technical cleanup.
Phase 2
Public site, registration, Stripe checkout, and confirmation emails.
Phase 3
Admin portal, inventory, pickup scanning, corporate orders, and exports.
Later
