# Terry Fox Run Singapore Approval Package

This folder contains meeting-ready approval materials for the proposed Terry Fox Run Singapore website.

## Files

- `index.html` - Start here. Links to the deck and proposal brief.
- `terry-fox-website-approval-deck.html` - 10-slide HTML presentation with static mockups.
- `terry-fox-website-proposal.html` - Print-friendly proposal brief for review.
- `terry-fox-website-proposal.md` - Editable Markdown source for the brief.
- `technical-stack-and-design-plan.html` - Print-friendly technical stack and design plan.
- `technical-stack-and-design-plan.md` - Editable Markdown source for the technical plan.

## How to Use

Open `index.html` in a browser, then choose the deck or proposal brief.

In the slide deck:

- Use right/left arrow keys to navigate.
- Use the on-screen controls if presenting from a browser.
- Print to PDF from the browser if a shareable PDF is needed.

For a Google Doc:

- Open `terry-fox-website-proposal.md`.
- Paste the contents into Google Docs.
- Or open `terry-fox-website-proposal.html` in a browser and copy the rendered document into Google Docs.
